# Airbnb Project (In Progress)

This project is a simplified Airbnb clone backend built using Node.js, Express.js, and MongoDB. The project is currently under development.

## Current Status

- Basic API routes created for core functionality
- Centralized error handling implemented for robust responses
- MongoDB integration set up for data storage

## Technologies Used

- Node.js
- Express.js
- MongoDB (Mongoose)

## Next Steps

- Implement user authentication and authorization (e.g., JWT)
- Complete remaining routes and features
- Improve validation and security
- Add testing for routes and error handling
